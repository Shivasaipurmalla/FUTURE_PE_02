# Project Overview

This project demonstrates how Prompt Engineering can be used to create authentic User-Generated Content (UGC) advertisements for a local café.

The goal is to generate marketing content that feels natural, relatable, and persuasive while encouraging customers to visit Brew Haven Café.

The project includes AI-generated:

- UGC Ad Scripts
- Social Media Hooks
- Call-To-Action (CTA)
- Instagram Captions
- Prompt Framework
- Prompt Iterations
- Prompt Outputs
