# Project Summary

## Project Title

AI Content Marketing using UGC Ads

---

## Business

Brew Haven Café

---

## Objective

To use Prompt Engineering techniques for generating authentic User-Generated Content (UGC) advertisements suitable for social media marketing.

---

## Deliverables

- UGC Ad Scripts
- Hooks
- CTAs
- Social Media Captions
- Prompt Framework
- Prompt Iterations
- Prompt Outputs

---

## Skills Learned

- Prompt Engineering
- AI Content Creation
- Digital Marketing
- UGC Advertisement Writing
- Social Media Marketing
- Marketing Psychology

---

## AI Tool Used

ChatGPT

---

## Conclusion

This project demonstrates how Prompt Engineering can be used to create engaging and effective marketing content for local businesses. Structured prompts and iterative improvements resulted in professional-quality UGC advertisements that are suitable for platforms such as Instagram Reels, Facebook Reels, and YouTube Shorts.
